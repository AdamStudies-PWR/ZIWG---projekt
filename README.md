## **Instrukcje**

Przygotowanie środowiska

```
python3 -m venv venv/
```

- Windows
    ```
    venv/Scripts/activate
    ```

- Linux
    ```
    . venv/bin/activate
    ```

```
pip3 install -r requirements.txt
```
